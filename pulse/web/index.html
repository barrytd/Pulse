Reads directly from C:\Windows\System32\winevt\Logs\ and runs every detection rule. No upload required.

How to get your Windows event logs

Three steps. Pick the path that matches how comfortable you are with the command line.

1
Open Event Viewer

Press Win + R, type eventvwr, press Enter.

2
Pick a log

In the left tree, open Windows Logs, then click System or Application. (Security will ask for admin.)

3
Save as .evtx

On the right, click Save All Events As..., choose .evtx as the file type, pick a name, save. Drop that file into Pulse.

1
Open a regular command prompt

No admin needed for the System and Application logs.

2
Export the last 7 days

Paste either command and hit Enter. The .evtx file lands in your current folder.

wevtutil epl System system.evtx /q:"*[System[TimeCreated[timediff(@SystemTime) <= 604800000]]]" /ow:true
wevtutil epl Application application.evtx /q:"*[System[TimeCreated[timediff(@SystemTime) <= 604800000]]]" /ow:true
3
Drag the .evtx file into Pulse

Close this tutorial, drop the file on the upload area, hit Scan.

1
Open an elevated command prompt

Click Start, type cmd, right-click Command Prompt, choose Run as administrator. The Security log requires this.

2
Export the Security log (last 7 days)

This is the log with logons, privilege escalation, account changes — the interesting stuff for Pulse.

wevtutil epl Security security.evtx /q:"*[System[TimeCreated[timediff(@SystemTime) <= 604800000]]]" /ow:true
3
Upload to Pulse

You can drop multiple .evtx files (System + Security + Application) at once — Pulse will scan them all in parallel.
