Real-time Windows threat detection,
without the SIEM bill.

Pulse parses Windows Event Logs, runs MITRE-mapped detections, and surfaces findings in a single dashboard. Run it on your laptop, drop it on a server, or hand it to your SOC — same code, same UI, same database.

Get the CLI → Join the waitlist
⚡

Live monitoring

Stream Event Log entries through detections in real time. Findings light up the dashboard the moment they happen — no batch lag.

⚙

MITRE-mapped rules

Every finding ties back to a specific ATT&CK technique. Reviewers see what happened, why it matters, and where it fits the kill chain.

📊

Score & trend over time

A single 0-100 security score per scan, plus a daily history line. Catch regressions before they become incidents.

🛡

One-click firewall

Block a malicious source IP straight from the finding drawer. Pulse pushes the rule to Windows Firewall in one round-trip.

📧

Email + Slack + Discord alerts

Threshold-tripped findings fan out to your channels of choice with a configurable cooldown so the queue stays signal, not noise.

🔍

AbuseIPDB enrichment

Source IPs come back with confidence-of-abuse scores, country, and ISP — cached for 24 hours so a noisy host doesn’t burn quota.
